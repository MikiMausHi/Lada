# My education FE #

### Development (localhost) ###

* yarn (first time)
* yarn build (first time)
* yarn start

### Test ###
* yarn
* yarn build
* yarn start:test

### Production ###
* yarn
* yarn build
* yarn start:prod or pm2 start pm2-prod.conf.js

### Branches ###
* master - девелоперская ветка
* prod - продуктовая ветка
